# Brain Games Concept Library By Subject

This file separates the current concept library by primary subject so the system can stay clear as the platform grows.

The goal is to avoid mixing math, language, science, and logic concepts too early inside the same design pipeline.

For now, the strongest product path remains math-first.

---

## Math

- Bakery Rush
- Pharmacy Counter
- Chef’s Fractions
- Bridge Load
- Galaxy Coordinates
- Trade Floor
- Archaeology Grid

### Math Strategy Note
These concepts are the closest fit for the current Math Game Studio OS and should remain the main focus until the first prototype path is proven.

---

## ELA / Language

- Animal Alphabet
- Spell Surgeon
- Newsroom Deadline
- Evidence Room

### Language Strategy Note
These concepts are promising, but they should not be mixed into the current math-first pipeline until the first math family is proven.

---

## Science

- Color Lab
- Forecast Station
- Dino Excavation
- Reef Survey
- Volt Circuit
- Diagnosis Protocol

### Science Strategy Note
These concepts show strong long-term platform potential, but they are future expansion, not current execution focus.

---

## Logic / CS

- Mission Decode
- Algorithm Sprint

### Logic / CS Strategy Note
These are especially promising for later expansion because they fit structured game thinking well, but they should remain separate from the current math-first evaluation batch.

---

## Cross-Disciplinary / Mixed Watchlist

These concepts may eventually span more than one subject area and should be watched as future broader platform candidates:

- Trade Floor
- Evidence Room
- Forecast Station
- Diagnosis Protocol

---

## Strategic Summary

The broad platform can remain:

**Brain Games Platform**
Games that build real skills for real careers

But the current execution path should remain:

**Math-first product system**

That means:
- test the strongest math concepts first
- prove one strong family first
- expand into other subjects later

Do not broaden the operating system before the first math prototype works.
