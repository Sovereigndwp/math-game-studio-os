## Learning materials:

[Permalink: Learning materials:](https://github.com/Roobyx/awesome-game-design?tab=readme-ov-file#learning-materials)

- [The Art of Game Design: A book of lenses](https://books.google.bg/books?id=LP5xOYMjQKQC&pg=PA382&lpg=PA382&dq=game+design+document+gallery&source=bl&ots=YafNpBP9yF&sig=ACfU3U34gaAKQryZ974IUWMCDf2co3SogA&hl=bg&sa=X&ved=2ahUKEwjiy9ylz6HqAhXjo4sKHdR6BGQQ6AEwAXoECAgQAQ#v=onepage&q=game%20design%20document%20gallery&f=false)
- [Game Maker's Toolkit](https://www.youtube.com/@GMTK) - Mark Brown's extensive video library analyzing game design principles
- [Lost Garden](https://lostgarden.home.blog/) - Daniel Cook's influential game design blog with essays and frameworks
- [The Door Problem](https://lizengland.com/blog/2014/04/the-door-problem/) - Classic game design essay by Liz England explaining what a game designer does
- [Designer Notes](https://www.designer-notes.com/) - Soren Johnson's game design podcast and blog
- [A Theory of Fun](https://www.theoryoffun.com/) - Raph Koster's influential book on game design (site with resources)
- [Asking Gamedevs](https://askingamedev.tumblr.com/) - Industry Q\&A collection from veteran game developers
- [Game Programming Patterns](https://gameprogrammingpatterns.com/) - Free online book by Robert Nystrom covering essential programming patterns for games
- [Game Design Patterns Wiki](https://virt10.itu.chalmers.se/index.php/Main_Page) - Academic collection from Chalmers University of Technology
- [Game Developer Deep Dives](https://www.gamedeveloper.com/keyword/deep-dive) - In-depth technical postmortems series
